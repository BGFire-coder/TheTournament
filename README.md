# The Tournament

A card based strategy game
## Installation

```-Download the ZIP folder, containing the project.```\
```-Decompress the files.```\
```-Open Unity Hub.```\
```-Click "Add" and then click "Add project from disk".```\
```-Find the decompressed folder and select it.```\
```-In the project go to Assest/Scenes and add all scenes to the hierarchy.```\
```-Unload all scenes but Login.```
